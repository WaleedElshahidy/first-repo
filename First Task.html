HTML (Hyper Text Markup Language) :


HTML describes the structure of a Web page.

HTML consists of a series of elements. HTML elements tell the browser how to display the content.

HTML elements are represented by tags

CSS (Hyper Text Markup Language):


CSS is a language that describes the style of an HTML document.

CSS describes how HTML elements should be displayed. This tutorial will teach you CSS
